Quân cờ thẳng hàng

Trên bàn cờ 8x8, chia thành 64 ô vuông, mỗi ô vuông có thể có 1 quân cờ.

  x x   x   x x
x   x   x   x  
x   x x x   x x
  x   x x x   x
x   x x   x x x
x         x    
  x     x     x
x     x     x  

Yêu cầu:

Tìm số quân cờ thẳng hàng lớn nhất, theo hàng ngang, dọc hoặc đường chéo.

Input:

từ tệp quanco.inp gồm có 8 dòng, mỗi dòng biểu diễn một hàng trên bàn cờ, tức là gồm 8 chữ số 0 (không có quân cờ) hoặc 1 (có quân cờ).

Output:

ghi ra tệp quanco.out một số duy nhất là số quân cờ thẳng hàng dài nhất

Ví dụ:

QUANCO.INP

01101011
10101010
10111011
01011101
10110111
10000100
01001001
10010010

QUANCO.OUT

5

Ghi chú:

Không có dấu cách trong tệp input
